# Get Me Food

An app for finding restaurants.

## Setup

Install dependencies

```shell
npm install
bower install
```

Create a config.js file in the project route

```javascript
module.exports = {
    yelp: {
        Consumer_Key: "Your Data Here",
        Consumer_Secret: "Your Data Here",
        Token: "Your Data Here",
        Token_Secret: "Your Data Here"
    }
};
```

## Usage

Build the project

```shell
grunt build
```

Start the project

```
grunt
```
