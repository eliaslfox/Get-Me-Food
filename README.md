# MissionBit-Fall-2015

My project for Mission Bit, Fall 2015 demo day.

## Installed Tools

### Grunt

Error Checking

```shell
grunt deploy
```

Launch

```shell
grunt launch
```

### Jshint

```shell
grunt lint
```

### Mocha

```shell
grunt test
```

### Apidoc

```
grunt apidoc
```

This saves the doc file to ./apidoc/index.html
